# Psychedelic p5.js Art

Made using https://p5js.org/ and deployed on Cloudflare Workers.

## Features

- **Kaleidoscope** — your image mirrored and rotated with rainbow color cycling
- **Golden Ratio Spirals** — fractal spirals based on the Fibonacci sequence
- **Sacred Geometry** — Flower of Life pattern pulsing at the center
- **Starfield** — parallax twinkling stars with depth layers
- **Shooting Stars** — random meteors streaking across the sky
- **Nebula Clouds** — soft, drifting cosmic gas formations
- **Energy Waves** — flowing sine waves with Perlin noise
- **Feedback Trails** — hypnotic tunnel effect from frame blending

## Controls

- **Mouse movement** — glowing cursor with particle trails
- **Click** — creates ripple effect and particle burst
- **Keys 3-9** — change kaleidoscope segments
- **R** — reset feedback trails
